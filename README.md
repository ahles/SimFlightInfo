# xpvuemap

A Electron-APP that listens to the position information sent by X-Plane (over UDP) and displays the current location on a map.

## Project setup
```
yarn install
```

## DEV Mode
```
yarn electron:serve
```

## BUILD App
```
yarn electron:build
```
