# whereismyplane

Electron-App that listens to the position information sent by X-Plane 11 on another computer (over UDP 49000) and displays the current location on a map.

## Project setup
```
yarn install
```

## DEV Mode
```
yarn electron:serve
```

## BUILD App
```
yarn electron:build
```

### Lints and fixes files
```
yarn run lint
```
